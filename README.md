# IDC — Indice de Dificultad de Contratacion

Herramienta web interactiva de [erecruit](https://erecruit.ca/) que permite a empleadores en Latinoamerica evaluar que tan dificil sera cubrir una posicion especifica, basandose en datos publicos del mercado laboral.

## Funcionalidades

- Calculo de dificultad basado en 5 variables del mercado laboral
- Cobertura de 7 paises: Mexico, Colombia, Brasil, Argentina, Chile, Peru, Ecuador
- 8 categorias de rol y 3 niveles de experiencia
- Analisis personalizado por inteligencia artificial
- Generacion de reporte PDF descargable
- Captura de leads integrada con Brevo

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- @react-pdf/renderer
- Vercel
